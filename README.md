# wau.my.id

Static personal website and blog for **wau.my.id**.

## Isi repo

- `index.html` — homepage utama
- `blog/` — blog static
  - `blog/index.html` — halaman daftar artikel
  - `blog/tentang-gw/` — artikel pertama

## Cara update

1. Edit file HTML yang diperlukan.
2. Commit perubahan ke repo ini.
3. Push ke GitHub.
4. Deploy ke web server.

## Catatan

Homepage utama dijaga tetap stabil, sedangkan artikel blog bisa ditambah lewat slug baru di folder `blog/`.
